Git

  • git init - Инициализирует новый репозиторий Git в текущем каталоге.
  • git clone - Клонирует удалённый репозиторий по указанному URL на локальную машину.
  • git status - Показывает текущее состояние рабочего каталога и индекса, включая изменения, готовые к коммиту.
  • git commit -m "" - Сохраняет изменения в локальном репозитории с сообщением о коммите.
  • git push - Отправляет коммиты из локального репозитория в удалённый репозиторий. Например, git push origin main.
  • git branch - Показывает список всех локальных веток. Используйте git branch для создания новой ветки.
  • git log - Показывает историю коммитов в текущей ветке.
  • git diff - Показывает различия между изменениями в рабочем каталоге и индексом, а также между коммитами.
  • git remote -v - Показывает список удалённых репозиториев и их URL.
